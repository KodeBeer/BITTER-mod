# septic.js

A preprocessor and webgl engine. 

 - INCLUDE(file)
 - INCLUDE(file, BASE64)
 - INCLUDE(file, TILE, x, y)
 - INCLUDE(file, TILESET)           // used to be TILEMAP
 - INCLUDE(file, WIDTH)             // echo width of image file
 - INCLUDE(file, FRAME, x, y)
